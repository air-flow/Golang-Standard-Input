# Golang-Standard-Input

汎用的に使用できる golang の標準入力作成

## DEMO

input : 10 12 1
output : [3]float64{10, 6, 3}

## Features

input, output ファイルにそれぞれわけて作成する。

## Requirement

### application

- Go 1.15.2

## Usage

```bash
git clone https://github.com/air-flow/Golang-Standard-Input.git
go input.go
```
